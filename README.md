# TorquePilot AI Conso

Dashboard local MVP pour suivre les projets, clients, statuts et consommations IA TorquePilot.

## Démarrage local

```bash
npm install
npm run dev
```

Puis ouvrir : http://localhost:3026

## Objectif MVP

- Vue d'accueil synthétique
- Suivi des projets
- Suivi clients
- Suivi consommation IA
- Notes rapides
